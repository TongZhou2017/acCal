# 关于 acCal

## 项目使命

**让学术回归纯粹，把时间还给科研。**

acCal 致力于解决国内学术会议信息分散、鱼龙混杂的痛点。我们提供一个开源、可订阅、且经过人工审核的纯净会议日历，帮助科研人员更高效地规划学术日程。

## 核心原则

1. **开源透明** - 所有代码和数据公开，社区共同维护
2. **质量优先** - 严格审核，拒绝掠夺性会议和商业培训
3. **社区驱动** - 通过 GitHub Issues 提交，自动化流程处理
4. **免费使用** - 永远免费，无广告，无商业推广
